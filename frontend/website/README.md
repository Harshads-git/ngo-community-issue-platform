# NGO-Driven Community Issue Reporting Platform

## Project Overview
A comprehensive frontend platform built with HTML5, CSS3, and Bootstrap 5 for managing community issues through AI-powered coordination between citizens, NGOs, volunteers, partner organizations, and donors.

## Technology Stack
- HTML5, CSS3, Bootstrap 5.3.2
- Bootstrap Icons 1.11.1
- Google Fonts (Outfit & Inter)

## Folder Structure
```
ngo-platform/
├── index.html              # Landing page
├── README.md               # Documentation
├── css/style.css          # Custom styles
└── pages/
    ├── public/            # Public pages
    ├── citizen/           # Citizen dashboard
    ├── ngo/              # NGO Coordinator dashboard
    ├── volunteer/        # Volunteer dashboard
    ├── partner/          # Partner dashboard
    ├── donor/            # Donor dashboard
    └── admin/            # Admin dashboard
```

## Features by Role

### Citizen
- Report issues with photos & location
- Track issue timeline
- View status updates
- Edit/withdraw reports

### NGO Coordinator
- View AI-classified issues
- Set priorities
- Assign volunteers/partners
- View analytics

### Volunteer
- View assigned tasks
- Accept/decline assignments
- Update progress with photos
- Mark tasks complete

### Partner Organization
- View assigned projects
- Update status
- Upload completion reports

### Donor
- View impact reports
- See contribution breakdown
- Donate to categories

### Administrator
- Manage all users
- Moderate issues
- View system logs
- System health monitoring

## Installation
1. Extract files
2. Open index.html in browser
3. Use login page to access dashboards
4. All features are UI-only (no backend)
